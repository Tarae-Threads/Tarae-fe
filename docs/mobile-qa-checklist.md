# 모바일 QA 체크리스트

출시 전 실기기에서 반드시 검증해야 하는 항목. 브라우저 DevTools 모바일 뷰가 아닌 **실제 디바이스**에서 수행.

## 대상 디바이스·브라우저

| 분류 | 모델 | 브라우저 | 비고 |
|---|---|---|---|
| iOS 소형 | iPhone SE (2~3세대) | Safari | 375×667, safe-area inset 없음 |
| iOS 대형 | iPhone 15 Pro / Pro Max | Safari | Dynamic Island, safe-area inset 크게 반영 |
| Android 플래그십 | Galaxy S23/S24 | Chrome | 기본 WebView도 1회 확인 |
| Android 보급형 | 저사양 기기 또는 throttled CPU | Chrome | 애니메이션/지도 렌더 끊김 확인 |

## 공통 체크

- [ ] 세로/가로 회전 시 레이아웃 깨짐 없음
- [ ] 다크모드 OS 설정 시 (현재 미지원) 가독성 유지
- [ ] 시스템 글자 크기 최대 설정에서 레이아웃 깨짐·컨트롤 사라짐 없음
- [ ] Safari/Chrome 주소창 toggle에 따른 뷰포트(`100vh` → `100dvh`) 대응
- [ ] 한국어 IME(삼성·구글) 입력 시 인풋 포커스·제출 동작 문제 없음
- [ ] 네이버 지도 제스처(핀치 줌, 드래그, 더블탭) 정상 동작
- [ ] PWA 설치(iOS "홈 화면에 추가") 후 실행 확인

## 화면별 체크

### 홈(`/`)
- [ ] Hero 섹션 이미지·카피 정렬
- [ ] TrendingPlaces / ActiveEvents 가로 스크롤 카드 — 터치 스와이프 부드러운지, `min-w-[280px]` 카드가 375px 화면에서 잘림 없이 스크롤되는지
- [ ] CategoryShortcut / RegionShortcut 버튼 — 터치 타겟 최소 44×44 이상
- [ ] InquiryBanner → 문의 모달 오픈 정상
- [ ] Footer의 이용약관·개인정보처리방침 모달 정상 오픈 및 스크롤

### 지도(`/map`)
- [ ] 초기 로딩 — 지도 블랭크 유지 시간, 스켈레톤 대체 확인
- [ ] 하단 `MobileBottomSheet` — 3단(closed/peek/full) 스냅포인트가 검색바·BottomNav와 **겹치지 않는지**
- [ ] 현재 위치 권한 팝업 — 거부 시에도 기본 위치로 동작
- [ ] 마커 클릭 → DetailPanel 바텀시트 전환 정상
- [ ] 딥링크 `/map?placeId=1`, `/map?eventId=1` 직접 진입 시 상세 자동 오픈
- [ ] 검색바 입력·결과 동기화, 현재 지역만 보기 토글 정상
- [ ] 카테고리·지역·거리 필터 적용 시 리스트·지도 동기화

### 제보 플로우
- [ ] 제보 모달 다단계 폼 — 이전/다음 버튼 동작, 키보드 올라와도 버튼 가려지지 않음
- [ ] 주소 검색(Kakao Postcode) 팝업 — 모바일 뷰포트에서 잘림 없음
- [ ] 전송 중 버튼 disabled, 중복 제출 방지

### 문의 / 리뷰
- [ ] 입력 필드 포커스 시 스크롤 자동 조정
- [ ] 이메일 키보드 타입 정상 (`type="email"`)
- [ ] 성공/에러 토스트가 BottomNav·BottomSheet에 가려지지 않음
- [ ] 리뷰 비밀번호 삭제 — 오답 시 에러 메시지 한국어 출력

## 접근성(A11y) 모바일 체크

- [ ] iOS VoiceOver / Android TalkBack 기본 스와이프 내비게이션 가능
- [ ] 포커스 이동 순서가 시각적 순서와 일치
- [ ] 모달 오픈 시 포커스가 모달 내부로 이동, ESC(VoiceOver escape) 가능
- [ ] 색상 대비 WCAG AA 기준 충족 (특히 칩·필터 색상 조합)

## 성능 체크 (Lighthouse 모바일)

| 지표 | 목표 | 측정 페이지 |
|---|---|---|
| LCP | ≤ 2.5s | `/`, `/map` |
| CLS | ≤ 0.1 | `/`, `/map` |
| TBT | ≤ 200ms | `/map` |
| Accessibility | ≥ 90 | 전체 |
| SEO | ≥ 90 | 전체 |
